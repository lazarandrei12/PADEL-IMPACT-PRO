# Padel Impact Pro

An immersive 2v2 multiplayer padel game delivering competitive gameplay, stunning graphics, and online multiplayer experiences.

Website: [padelimpact.games](https://padelimpact.games/)

## Overview

Padel Impact Pro is a modern sports simulation game that captures the intensity and precision of professional padel tennis. Play solo, compete with friends, or challenge players worldwide in dynamic 2v2 matches.

### Key Features
- 2v2 Multiplayer Gameplay - Team-based competitive matches
- Ranking System - Climb the leaderboards
- Real-World Inspired Courts - Play in iconic locations
- Stunning Graphics - Immersive visual experience
- Precision Mechanics - Feel every shot and serve
- Cross-Platform Play - Play with friends on any device

## Tech Stack - Website

- Framework: React
- Build Tool: Vite
- Styling: Tailwind CSS + Framer Motion (Animations)
- Hosting & CI/CD: Netlify
